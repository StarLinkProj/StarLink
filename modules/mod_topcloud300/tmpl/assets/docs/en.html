Cloud links can have a variety of fonts and colours








There are preset colour themes for the links.
You can also select custom colours.

You can have a cloud made of simple text or make use of the
graphics of the module.


Patterned backgrounds are available. If you wish to use
other backgrounds it is quite easy.
Simply place png graphics files in the backgrounds folder at:
mod_topcloud300/tmpl/assets/images/backgrounds
The items will then appear in the dropdown menu at the left.


Borders,shadow and background colours are easily selected
using the colour widgets.

The sources for the cloud match the Joomla search plugins.
Clicking on item in the cloud has the same effect as typing
it into Joomlas integral search.










Unwanted individual items may be excluded Items may also be added.


Adding an item multiple times will increase its font-size in the cloud.


The numerical size and starting point of the cloud can be set








The link sources above can be restricted by language


Additional characters from other languages can be entered here.
If you need to you can add whole alphabets.


The search results page can be chosen

If you have more than one cloud module on a page a ID number is set to
prevent the CSS from interfering. This is usually set automatically

Cloud items may be either individual words or phrases

Cloud items can be shuffled when a page loads.

Unregistered users can be restricted to links which only relate to publicly available content.








The overall module size and position can be set by these basic CSS
controls.









Also further scripts and styling may be added to the <head> section of the page. This can be useful if you need to modify the effect styling of your template or other page items may have on the cloud
